# Archive

This folder keeps longer project notes that are useful for history, but not needed for a quick review of the repo.

## Contents

- `build_logs/`: phase notes and validation logs from the build process
- `reference/`: setup notes, commands, source documentation, and manual SQL prototypes
- `supporting/`: detailed cleaning rules and KPI dictionary notes
- `tableau/`: Tableau planning and rebuild notes

The main project path is still the root `README.md`, `ARCHITECTURE.md`, `KPIS.md`, `dbt_project/`, `ingestion/`, and `dashboard_preview/`.
