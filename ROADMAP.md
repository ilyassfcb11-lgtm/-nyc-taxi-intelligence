# Roadmap

This roadmap is a target, not a guarantee. If a phase takes longer than expected, the project should reduce scope before adding unnecessary complexity.

## Phase 0: Setup

Target: 1 to 2 days.

Deliverable:

- Repository structure.
- Local Python environment.
- GCP and BigQuery plan.
- Starter documentation.

## Phase 1: Ingestion

Target: 3 to 4 days.

Deliverable:

- Download script for selected TLC files.
- Raw Yellow Taxi data loaded into BigQuery.
- Raw Taxi Zone Lookup loaded into BigQuery.

## Phase 2: Cleaning and KPIs

Target: 3 to 4 days.

Status: complete.

Deliverable:

- Cleaned trip tables.
- Fact and dimension tables.
- KPI definitions.
- Cost-conscious summary tables for analysis.
- Phase 2 summary and interview notes.

## Phase 3: Visualization

Target: 3 to 4 days.

Status: started.

Deliverable:

- Tableau dashboard pages.
- First visible portfolio-ready version.
- Initial business insights and recommendations.
- Tableau plan and workbook notes.

## Checkpoint 1

Target: about 2 weeks.

Deliverable:

- A working end-to-end MVP from raw data to dashboard.

## Phase 4: dbt Modeling

Target: 5 to 7 days.

Status: complete.

Deliverable:

- Staging models.
- Fact and dimension models.
- Mart models for dashboard use.

## Phase 5: Testing and Data Quality

Target: 3 to 4 days.

Status: complete for MVP.

Deliverable:

- dbt tests.
- Data quality checks.
- Data quality documentation.

## Phase 6: CI/CD and Polish

Target: 2 to 4 days.

Status: started.

Deliverable:

- GitHub Actions workflow.
- Final documentation.
- Interview preparation notes.

## Checkpoint 2

Target: 4 to 6 weeks, with 6 to 8 weeks acceptable if needed.

Deliverable:

- A professional portfolio project ready for job applications and interviews.
